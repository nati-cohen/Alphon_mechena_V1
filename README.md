
# אלפון תלמידים (Student Directory) - מכינת בית אל

אפליקציית ווב מתקדמת לניהול וצפייה ברשימת תלמידים, מבוססת Google Sheets ופועלת כ-PWA (ניתנת להתקנה על מסך הבית).

## תכונות עיקריות

- **סנכרון חי:** הנתונים נטענים בזמן אמת מגיליון Google Sheets.
- **ימי הולדת עבריים:** הצגת חוגגי החודש עם סימון מיוחד ברשימה ודף ייעודי.
- **ממשק PWA:** אפשרות להתקנה ב-iOS ובאנדרואיד כיוונית "אפליקציה" לכל דבר.
- **חיפוש וסינון:** חיפוש מהיר לפי שם/טלפון וסינון לפי כיתות.
- **פעולות מהירות:** חיוג ישיר, שליחת WhatsApp והעתקת מספר בלחיצה.
- **מצב כהה/בהיר:** תמיכה מלאה בעיצוב מותאם אישית למשתמש.

## מה חדש? (What's New)

### גרסה 1.0.1
- **מערכת ימי הולדת:** נוספה תמיכה בקריאת תאריכי ימי הולדת עבריים מהגיליון.
- **תצוגת ימי הולדת:** 
  - הוספת דף "חוגגים החודש" נגיש מהתפריט הצדדי.
  - סימון תלמידי יום ההולדת ברשימה הכללית עם אייקון עוגה וכיתוב "מזל טוב".
  - תצוגה נקייה של התאריך (יום וחודש בלבד) ללא שנת לידה.
- **תיקוני תאריכים:** המרת תאריכי הכותרת לגימטריה מלאה (למשל: ג' טבת).
- **שיפורי UI:** עדכון התפריט הצדדי (Sidebar) ושיפור מהירות הטעינה.

## פיתוח וטכנולוגיות

- **Framework:** React 19 + TypeScript.
- **Styling:** Tailwind CSS.
- **Routing:** React Router 7.
- **Data Parsing:** PapaParse (CSV).
- **Deployment:** Vercel / GitHub Pages.

## חיבור לנתונים

כדי לעדכן את המקור, יש לשנות את `GOOGLE_SHEET_CSV_URL` בקובץ `constants.ts`. הגיליון צריך לכלול את העמודות: `full_name`, `phone_number`, `image_url`, `class`, `notes`, ו-`יום הולדת`.
