# אלפון תלמידים (Student Directory)

אפליקציית ווב לניהול וצפייה ברשימת תלמידים, מבוססת Google Sheets.
האפליקציה נבנתה באמצעות React, TypeScript ו-Tailwind CSS.

## תכונות

- **חיבור ל-Google Sheets:** הנתונים נטענים בזמן אמת מקובץ אקסל שיתופי.
- **עיצוב מובייל-פירסט:** מותאם באופן מושלם לטלפונים ניידים.
- **חיפוש וסינון:** אפשרות לחפש לפי שם/טלפון ולסנן לפי כיתות.
- **פעולות מהירות:** חיוג, שליחת הודעת וואטסאפ והעתקת מספר בלחיצת כפתור.
- **תמיכה מלאה בעברית (RTL).**

## הרצה מקומית

1. התקן את התלויות:
   ```bash
   npm install
   ```

2. הרץ את שרת הפיתוח:
   ```bash
   npm run dev
   ```

3. פתח את הדפדפן בכתובת שמופיעה בטרמינל (בדרך כלל http://localhost:5173).

## הגדרות

כדי לחבר את האפליקציה ל-Google Sheet שלך, ערוך את הקובץ `constants.ts` והחלף את המשתנה `GOOGLE_SHEET_CSV_URL` בקישור ה-CSV שלך.

## פריסה (Deployment)

הפרויקט מוכן לפריסה ב-GitHub Pages, Netlify או Vercel.
